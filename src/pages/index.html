The full-stack web is here

Greenwood embraces web standards from the ground up, empowering your stack from front to back, to run anywhere the web can run.

Intuitive hybrid, file-based routing allows for static or server rendered pages side by side. Start from just an index.html file if you need.

Server rendered pages are themselves just custom elements, that of course can consume other web components.

Web Components are full-stack in Greenwood and so the same definition can be used on the client or the server! And with WCC, you are free to use Light DOM or Shadow DOM.

Include API Routes for easy client side fetching, including rendering web components. Don't be scared, you can even return HTML!

Now, just build and go!
