GreenwoodJS

Intuitive hybrid, file-based routing allows for static or server rendered pages side by side.

Server rendered pages are themselves just custom elements, that of course can consume other web components.

That can of course consume web components. Web Components are full-stack in Greenwood! And with WCC, you are free to use Light DOM or Shadow DOM.

Include API Routes for easy client side interaction, including rendering web components. Don't be scared, you can even return HTML!

Now, just build and go!
