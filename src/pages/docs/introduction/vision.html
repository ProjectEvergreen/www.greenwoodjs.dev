Vision

Greenwood's goal is to make web development easier and more accessible by leveraging the web standards we all know and love and applying them diligently across the stack from front to back. We believe that by being as faithful to the web as possible, Greenwood projects can rely less on bespoke implementation details and thus afford developers with reduced overhead to build and streamlining time to ship. In this way, Greenwood aims to distinguish itself (pedantically perhaps) as a workbench for the web, rather than a framework.

By doubling-down on the web and infusing it into the heart Greenwood's philosophy, and technical design, as well as participating in community group efforts to extend the web when needed, Greenwood aims to make you more productive in building sites and full-stack apps that feel just as seamless to build today as it was the day you wrote your first index.html page. So with MDN as our companion, let's all see how far we can take the web together!
